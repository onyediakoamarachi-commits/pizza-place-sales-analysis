# pizza-place-sales-analysis
This project analyzes one year of sales data from a fictional pizza place. The dataset includes information on orders, pizza types, sizes, prices, quantities, and timestamps.

## Objectives

- Calculate total revenue, total orders, and total quantity sold.
- Identify peak hours and best-selling pizzas.
- Analyze sales trends by day of the week and by month.
- Detect underperforming pizza types.
- Explore seasonal trends in sales to inform business decisions.

- *Total Revenue:* $817,860.05
- *Peak Hours:* The busiest hours are around dinner time (e.g., 19:00).

## Tools Used

- Python (pandas, numpy, matplotlib)
- Jupyter Notebook

## Usage

1. Clone the repository.
2. Place the CSV files (Orders.csv, Order Details.csv, Pizzas.csv, Pizza Types.csv) in the project folder.
3. Open the Jupyter Notebook and run all cells to reproduce the analysis and visualizations.
